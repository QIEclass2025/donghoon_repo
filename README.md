# 윷놀이 게임

파이썬으로 만든 포켓몬 윷놀이 게임입니다.

## 게임 규칙

### 1. 게임 목표
- 두 명의 플레이어가 각각 4개의 말을 가지고 게임을 시작합니다.
- 자신의 말 4개를 모두 먼저 완주시키는 플레이어가 승리합니다.

### 2. 윷 던지기 결과
- **도**: 1칸 앞으로 이동
- **개**: 2칸 앞으로 이동
- **걸**: 3칸 앞으로 이동
- **윷**: 4칸 앞으로 이동 + 한 번 더 던지기
- **모**: 5칸 앞으로 이동 + 한 번 더 던지기
- **빽도**: 1칸 뒤로 이동

### 3. 특별 규칙
- **한 번 더!**: 윷이나 모가 나오면 한 번 더 윷을 굴릴 수 있습니다.
- **잡기**: 상대방 말이 있는 칸에 도착하면 상대 말을 잡고 처음 위치로 보낼 수 있습니다. 말을 잡으면 한 번 더 윷을 굴릴 수 있습니다.
- **업기**: 자기 말이 있는 칸에 도착하면 두 말을 겹쳐서 함께 움직일 수 있습니다.
-**지름길**: 판의 각 모서리와 중앙 부분에서는 지름길을 이용할 수 있습니다.

## 조작 방법

1.  **게임 시작**: `python yutnori_gui_real_final.py` 파일을 클릭하여 게임을 실행합니다.
2.  **윷 굴리기**: 자신의 턴이 되면 '윷 굴리기' 버튼을 눌러 윷을 던집니다.
3.  **말 움직이기**:
    - 윷을 던지면 이동할 수 있는 결과가 버튼으로 표시됩니다.
    - 움직이고 싶은 자신의 말을 클릭합니다. (대기 중인 말 또는 판 위의 말)
    - 사용하고 싶은 이동 버튼을 클릭하여 말을 움직입니다.
4.  **턴 넘기기**:
    - 이동할 수 있는 횟수를 모두 사용하면 자동으로 상대방에게 턴이 넘어갑니다.
    - 만약 '빽도'가 나왔지만 판 위에 움직일 말이 없는 경우, '턴 넘기기' 버튼이 나타나며 이를 통해 턴을 넘길 수 있습니다.

## 승리 조건
- 4개의 말이 모두 판을 완주하여 도착점에 들어오면 게임에서 승리합니다.
